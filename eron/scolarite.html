Inscription

Pour inscrire votre enfant au Collège Privé Les ERONS, veuillez suivre ces étapes :

  • Remplir le formulaire d'inscription disponible en ligne.
  • Soumettre les documents requis : certificat de naissance, bulletins scolaires précédents, et photo d'identité.
  • Fixer un rendez-vous pour un entretien avec l'administration.
  • Recevoir la confirmation d'inscription par email.

Horaires Scolaires

Primaire

Les horaires pour les élèves du primaire sont les suivants :

  • Du lundi au vendredi : 08h00 - 12h00 (Matin) / 13h30 - 15h30 (Après-midi)
  • Récréations : 10h00 - 10h15 / 15h00 - 15h15
  • Pause déjeuner : 12h00 - 13h30

Secondaire

Les horaires pour les élèves du secondaire sont les suivants :

  • Du lundi au vendredi : 08h00 - 12h00 (Matin) / 13h30 - 17h00 (Après-midi)
  • Récréations : 10h00 - 10h15 / 15h00 - 15h15
  • Pause déjeuner : 12h00 - 13h30

Ressources Disponibles

Pour le Primaire

Les élèves du primaire ont accès aux ressources suivantes :

  • Bibliothèque : Une riche collection de livres adaptés aux jeunes lecteurs.
  • Salle Informatique : Ordinateurs modernes avec des logiciels éducatifs adaptés.
  • Matériel Pédagogique : Outils et ressources pour faciliter l'apprentissage.

Pour le Secondaire

Les élèves du secondaire bénéficient des ressources suivantes :

  • Bibliothèque : Une vaste collection de livres, revues et ressources numériques.
  • Salle Informatique : Équipements modernes avec accès à Internet pour la recherche et les projets.
  • Laboratoires : Espaces dédiés aux sciences pour des expériences pratiques.
  • Support Académique : Séances de tutorat et aide aux devoirs disponibles.
